# Leon A

## Hardcore Progression

### Phase 1: Reach the RPD

1. Gas station sequence.
2. Run through Raccoon City.
3. Reach the police station.
4. Meet Marvin.
5. Receive the objective to collect 3 medallions.

### Phase 2: Early RPD

1. Solve the Lion Statue puzzle.
2. Explore East and West Wings.
3. Obtain Bolt Cutter and Spade Key.
4. Open previously locked routes and create shortcuts.

### Phase 3: Upper Floors

1. Reach the Lounge.
2. Solve the Unicorn Statue.
3. Encounter your first Licker.
4. Continue unlocking paths through the station.

### Phase 4: C4 Explosion

1. Obtain the Detonator and Battery.
2. Return to West Storage Room.
3. Blow open the C4 barrier.
4. Access Maiden Statue.
5. Collect the third medallion.

### Phase 5: Exit the RPD

1. Return to Main Hall.
2. Insert all three medallions into the Goddess Statue.
3. Open the underground passage beneath the RPD.

### Phase 6: Underground Facility

1. Use the elevator.
2. Descend the stairs.
3. Reach the machinery area.
4. Push the blocking bookshelf/shelf. This action triggers the first Birkin fight.
